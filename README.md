
# BStackDemo Automation Framework

## Project Overview

This project is an automation testing framework developed for the BrowserStack Demo E-Commerce website using Selenium WebDriver with Java.

The framework is designed using the Page Object Model (POM) design pattern and includes reusable utilities, reporting, screenshots, waits, and cross-browser support.

---

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Extent Reports
- WebDriverManager

---

## Framework Features

- Page Object Model (POM)
- Cross Browser Support
- ConfigReader Utility
- WaitUtils Utility
- Screenshot Capture on Failure
- Extent HTML Reports
- Dynamic XPath Handling
- Reusable Methods
- TestNG XML Execution
- Explicit Waits

## How To Run The Project

1. Clone the repository
2. Open project in Eclipse
3. Update Maven dependencies
4. Run testng.xml file
